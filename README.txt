Projet Snake pour l'UE LIF7:

(Auteurs: Roberto Medina Bonilla, Yoann Jouvent)

/!\ Générer la documentation pour plus d'informations! /!\

Snake dévellopé avec la bibliothèque SDL.

====== Pour installer SDL: =======

..$ sudo apt-get install libsdl1.2-dev libsdl-ttf2.0-dev

====== Pour compiler: ========

../svn$ make

ou ouvrir le projet Snake.cbp et appuyer sur F9

../svn$ codeblocks Snake.cbp &

====== Pour éxécuter depuis le terminal: =======

../svn$ ./bin/snake_gfx

====== Pour générer la documentation depuis le répertoire svn du projet pour les sources : =====

../svn$ doxygen doc/snake.dox

** Pour ouvrir la documentation:

../svn$ firefox doc/html/index.html &
